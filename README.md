# ibaraki-university-mechanical-engineering-coursework

[茨城大学工学部機械工学科](https://www.mechsys.ibaraki.ac.jp/)の学生に向けて、講義ノート、課題、レポート、実験資料、コードを科目別に整理した学習支援リポジトリです。

これから履修する学生の予習、履修中の整理、試験前の復習に使いやすいように、分野ごとに資料をまとめています。

## 概要

このリポジトリには、機械工学科で扱う主要科目に加えて、制御系、情報系、数学系、実験系の学習資料を収録しています。
各科目の資料フォルダ名は、原則として `note`、`report`、`code` に統一しています。

## 収録内容

- 講義ノート
- 演習・課題・レポート
- ソースコードと実装例

## ディレクトリ構成

```text
course/
├── 制御系/            # デジタル信号処理、メカトロニクス、ロボット工学、制御工学
├── 四力系/            # 材料力学、機械力学、流体力学、熱力学
├── 実験（情報コース）/ # CAD、写真、組込みプログラム、参考資料
├── 情報系/            # アルゴリズム、プログラミング、人工知能、画像情報処理
├── 数学系/            # モデル化、フーリエ解析、線形代数など
├── 数理基礎/          # 微積分学、力と運転 など
└── 設計系/            # 機構学 など
```

## 使い方

1. 学びたい分野のフォルダを開く
2. `note`、`report`、`code` などの下位フォルダを確認する
3. 個別 README がある科目は、先に概要を読む

## 注意

- 個人の学習記録を整理した資料であり、内容の正確性を保証するものではありません。
- 科目名やフォルダ構成は継続的に整理する可能性があります。
- 年度や担当教員によって、講義内容や課題の形式が異なるので、各自で確認してください。

## Contributing

このリポジトリは、茨城大学機械工学科の学生が自由に利用・編集できることを目的としています。
もし、より良い資料やコードがあれば、Pull Request を歓迎します（学生番号でブランチを分けてください）。
ただし、内容の正確性や著作権には十分注意してください。
